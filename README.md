# MazeRush

MazeRush is a simple tile-based maze game written in **SWI-Prolog**.  
Created by **Vidám Szabolcs** and **Tunya Lénárd-Sándor**.

## Overview

The goal of MazeRush is to escape from the randomly generated maze as fast as possible.
